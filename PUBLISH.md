# Публикация в PyPI с использованием Poetry

Данный документ описывает процесс публикации пакета python-allure-client в Python Package Index (PyPI) с использованием Poetry.

## Сборка пакета

Для сборки дистрибутивных файлов пакета выполните команду:

```bash
poetry build
```

Эта команда создаст директорию `dist/` с файлами `.whl` и `.tar.gz`.

## Публикация в PyPI

### Настройка токена PyPI (рекомендуется)

Вместо использования имени пользователя и пароля рекомендуется использовать токен API:

```bash
# Для PyPI
poetry config pypi-token.pypi ваш-токен-здесь
```

### Публикация в PyPI

Для публикации пакета в PyPI выполните команду:

```bash
# Публикация в продакшен PyPI
poetry publish

# Или сначала в Test PyPI для тестирования
poetry publish --repository testpypi
```
